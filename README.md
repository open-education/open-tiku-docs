# open-tiku-docs

开放题库产品说明文档

文档提交后自动发布到 [open-tiku-docs](https://open-education.github.io/open-tiku-docs/)

## 文档位置

文档存储在 docs 目录中, 注意添加头信息即可.

## 文档转换

针对手上的 word 等文档请使用其它工具比如 [Word to Markdown](https://word2md.com/) 转化为 markdown 文档.

## 图片存储

文档关联的图片请存储在 `assets/images/` 目录中, 根据自己的需要创建文件夹, 使用需要该格式

```
![image]({{ '/assets/images/catalog/add-7-1.png' | relative_url }})
```
